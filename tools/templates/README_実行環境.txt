NS-ENGINE ゲーム実行環境
========================

【実行方法】
game.exe をダブルクリックして起動してください。

【必要ファイル】
- game.exe            : ゲーム本体
- assimp-vc143-mt.dll : 3Dモデル読み込みライブラリ
- assets/             : ゲームアセット（シェーダー、テクスチャ、モデル、マテリアル）

【動作環境】
- Windows 10/11 64bit
- DirectX 11対応GPU
- Visual C++ 再頒布可能パッケージ (2022)

● ゲームの操作方法：                                                                                                             
                                                                                                                                 
  タイトル画面                                                                                                                      
  - Space / Enter → ゲーム開始                                                                                                   
                                                                                                                                 
  ゲーム中
  ┌────────────┬────────────────────────┐
  │    操作    │          キー          │
  ├────────────┼────────────────────────┤
  │ 移動       │ W/A/S/D または ↑/←/↓/→ │
  ├────────────┼────────────────────────┤
  │ ジャンプ   │ Space                  │
  ├────────────┼────────────────────────┤
  │ カメラ回転 │ マウス左/右ドラッグ    │
  └────────────┴────────────────────────┘
  - ゴールに触れるとクリア
  - 穴に落ちるとリスタート

  リザルト画面

  - Space / Enter → タイトルに戻る
