# Security Scan

`$ARGUMENTS` が指定された場合はそのファイル/ディレクトリを対象にします。
指定がない場合は `source/` 全体を対象にします。

## Task
コードベースのセキュリティ脆弱性を分析:

- SQLインジェクションリスク
- XSS脆弱性
- 露出した認証情報
- 安全でない設定
- 認証/認可の欠陥
- バッファオーバーフロー（C++）
- メモリリーク
- 未初期化変数の使用

`$ARGUMENTS` で対象が指定されている場合は、そのパスを重点的にスキャンしてください。

ソースコードを検索して問題を特定し、修正案を提示してください。
